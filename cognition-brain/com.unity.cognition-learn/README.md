# com.unity.cognition-learn

ML-Agents is a Unity package that allows users to use state-of-the-art machine learning to create intelligent character behaviors in any Unity environment (games, robotics, film, etc.).

Please refer to the [Unity Package Documentation](https://docs.unity3d.com/Packages/com.unity.cognition-learn@latest) for comprehensive installation, usage guides, tutorials, and API references.
